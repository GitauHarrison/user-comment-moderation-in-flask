Instructions on How to Use This App Locally

Posting Comments

  • Click on the "Articles" dropdown link and select one article

  • Post several comments on the selected article (say article 1) page using dummy credentials

Log in as Admin

  • Click on the Admin link in the top-right navbar to access all user comments posted on the blog

  • You will need to register first, so create a dummy admin account

  • Log into the new admin account. You should be able to see all user comments

Allow/Delete Posts

  • Choose an article among the cards in the Admin page by clicking "Take Action" button

  • Click Delete link to delete a user's comment

  • Click Allow to allow a user's comment to appear in the selected article page

  • Navigate to the selected article (say article 1) page to see all "Allowed" comments

Another try

  • Select another article (say article 2)

  • Post several comments on the selected article page using dummy credentials

  • Click on the Admin link to access all user comments posted on the blog

  • Click Delete link to delete a user's comment

  • Click Allow to allow a user's comment to appear in the Home page

  • Navigate to the selected article (article 2) page to see all "Allowed" comments

Final Result

  • You can log out as an admin and your allowed articles should persist in the selected articles' pages
